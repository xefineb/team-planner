## What is this?

**Team Topologies Planner** is a visual, interactive tool designed to help organizations design and visualize their team structures according to the [Team Topologies](https://teamtopologies.com/) framework.

It allows you to map out your organization's cognitive load, interactions, and value streams in a way that static diagrams cannot.
